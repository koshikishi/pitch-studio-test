# Pitch Studio Test
The test project for Pitch Studio.

* 1 page (product page)
* Created with HTML5 + CSS3 + JS ES6
* Used preprocessors: Pug + SCSS
* Built with Gulp + Webpack
* Responsive layout optimized for 320px, 720px, 1360px
* Flexbox, grid layout, web-optimized and responsive WebP images
* Support for displays with higher pixel density (Retina)
* Fonts: Rajdhani
* Tested in Chrome, Firefox, Safari, Opera, Edge, IE11
* Valid W3C markup
* 99% Google PageSpeed

GitHub Pages build: https://koshikishi.github.io/pitch-studio-test/
